# URL-Shortener-Local-
A local web-based URL shortener that:  Converts long URLs into short codes Redirects short URLs to original links Stores mappings locally (SQLite)  Built using Flask + SQLite.

## Description
A simple local URL shortener built using Flask and SQLite.

## Features
- Shortens long URLs
- Redirects short links
- Stores data locally

## Run
python app.py
